## Demo Template
Шаблон тестового сайта

## Установка 

1. Скачать и разархивировать архив с файлами.
2. С помощью FTP-клиента (рекомендую [FileZilla](https://filezilla-project.org/)) выгрузить все файлы в корень хостинга.
